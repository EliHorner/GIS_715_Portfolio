Map 1: Harmonious Colors

This map shows the Allentown/Bethlehem area of Pennsylvania, at the same scale as the larger scale map from the Scale Lab. It also shows the same features as that map, ones related to transportation, waterbodies, and political boundaries. This map was made to use a harmonious color scheme. The map should provide enough contrast between features to tell them apart, but uses somewhat similar color to create a scheme that is pleasant to look at.

Map 2: High Contrast Colors

This map shows the same area as the first map, with the same features displayed, but focuses on using a color scheme that provides maximum possible contrast between features. This map does not look as visually appealing as it could, as it instead is focused on simply providing hihg contrast between the various map features. The features are very easy to tell apart, but the map is not as pleasant to look at.
