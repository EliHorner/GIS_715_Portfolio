Scale Lab (ArcPy)

This section shows maps made to explore the same area at different scales. The visualizations in this section were created using ArcPy and Jupyter Notebooks. ArcPy is a Python site package that provides a useful way to access many of the toos of ArcGIS Pro in a Python interface.

Color Lab (ArcPy)

This section includes visualizations of the same maps with different color schemes, to explore the effects of color on perception and effective communication through visuals. The visualizations in this section were created using ArcPy and Jupyter Notebooks.

Glyphs Lab (ArcPy)

This section includes maps that use glyphs to convey information. Glyphs are a visualliztion tool that allows information to be presented through small symbols. The visualizations in this section were created using ArcPy and Jupyter Notebooks.

Small Multiples Lab (ArcPy)

This section includes visualizations of small multiples, a way to convey inforamtion from multiple small maps or other visualizations at once. They can also be used to produce animations of maps. The visualizations in this section were created using ArcPy and Jupyter Notebooks.
